# Sinew — fork de Julien

Ce fork garde Sinew proche de l’original, avec des optimisations conçues pour un usage simple, rapide et entièrement automatisé.

## Ajouts & Optimisations

- **Option de Langue :** Option `English / Français` dans `Settings > About`.
- **Mode Power User (Non-coder) :** L'assistant gère lui-même toute la technique (mises à jour, Git pull, commits, pushes, builds) en arrière-plan. Ses réponses sont ultra-concises et simples, sans jargon.
- **Multi-comptes OpenAI :** Gestion et connexion multi-comptes automatisées avec raccourcis intelligents (5.5 XHigh Fast).
- **Intégrations :** Extension Sinew Chrome Bridge et installeur dynamique inclus.
- **Automatisation MCP :** Enregistrement automatique des serveurs de connaissances (MCP) dans la base de données.
- **Logs simplifiés :** Sauvegarde automatique des logs de l'application dans `desktop-app.log`.

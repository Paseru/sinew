# Changelog Julien

- **Mise à jour v0.1.25 :** Synchronisation complète avec la version officielle de Sinew.
- **Git Windows :** Notre correction temporaire a été retirée au profit de la nouvelle correction officielle de Sinew v0.1.25 (plus robuste et dynamique).

- **Logs Hybrides :** Intégration d'un système de logging permanent et hybride dans `desktop-app.log`.
- **Option de Langue :** Option `English / Français` dans `Settings > About`.
- **Règle Globale :** Réponses simples, concises, adaptées au profil power-user.
- **Auto-update :** Désactivation de l'auto-update officiel dans le fork pour empêcher l'écrasement de nos ajouts.
- **Scripts :** Ajout de scripts PowerShell pour la synchronisation, sauvegarde et build automatiques.
